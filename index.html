Hosting di GitHub Pages = aplikasi bisa dibuka dari HP mana saja, gratis, tanpa login.

1. Buka github.com → login akun kamu
2. Buka repo tgofreshindonesia-hub → folder yang ada
3. Upload file tgofresh-realtime.html yang sudah didownload
4. Rename jadi realtime-panen.html (atau sesuai yang sudah ada)
5. Commit → GitHub Pages otomatis update dalam ~1 menit
6. Buka dari HP: https://tgofreshindonesia-hub.github.io/realtime-panen/
⚠️ Setiap kali file diupdate di Claude, perlu upload ulang ke GitHub. Data tetap aman karena tersimpan di Google Sheets dan localStorage HP masing-masing.

Paste kode ini ke Google Apps Script. Sudah support 3 sheet otomatis: Batch Panen, Transaksi, Rekap Harian.

// Lihat file tgofresh-apps-script.gs yang sudah didownload
// atau copy dari: deploy ulang dengan kode terbaru v2.0
// Sheet otomatis: Batch Panen | Transaksi | Rekap Harian
// doPost() menerima type: batch | update_status | test
// doGet()  → ping test dari browser
